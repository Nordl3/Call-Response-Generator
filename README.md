# Call / Response Generator

Browser-based call and response MIDI phrase generator for acid, techno and house.

It generates short paired phrases, lets you preview them in the browser, and exports MIDI files for the call, response, combined pattern, or both separate files.

## Features

- Call / response phrase generation
- Style-based phrase shaping
- Phrase setup controls for key, scale, steps, density, contour, repetition and note pool
- Separate call and response editing actions
- Shift call / response left and right
- Browser playback with panic / restart
- Metronome with level control
- Sound engine selection for quick auditioning
- MIDI export:
  - Combined
  - Call only
  - Response only
  - Layered
  - Both separate files
- Project JSON save / load with settings and pattern data

## Notes

This is a plain HTML/JS tool intended to run directly in the browser or via GitHub Pages.

No build step is required.

## Files

- `index.html` — the app

## Usage

Open the page in a browser, generate or edit a phrase pair, audition it, then export MIDI or save the project JSON.
